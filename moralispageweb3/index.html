About The Project CAMPUS

This platform allows the creation of mixed reality experiences, which can be viewed through smart glasses, these experiences are created from our campus project where the user with a basic use of unity can create these experiences and upload them to our platform using ipfs, to later be able to generate the nft, then sell it through our market, built on the Avalanche blockchain and Moralis.
